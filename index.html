Core philosophy: Non-combat → unlock efficient combat → farm ethereal eyes → Haste Tea (50% non-combat speed) → turbocharge non-combat → repeat. Rush eyes every run. Haste Tea is too effective not to use.

Main engine Cedar → Oil → Cores → Seeker → Everything
1
Cedar woodcutting (90%)
2 logs/action · pine knots 7.9% · powpow ~5.7% · salmonberries ~31% · satchels · nettles via landkeeping · Community doubles everything. Mask+Seeker only = ~14% rare rate · full Verdant 4pc = ~26%.
2
Pine knots → oil → enchanting offerings → all 4 cores
600 pine knots → 15k oil (25/knot via Electricity). Offer anything in bulk immediately — enchanting level doesn't affect core rates. Requires Enchanting 60 (52 with mask + merchant potion).
3
A few Seeker Potions to bootstrap
Best early source: fishing merchant then potion merchant. Stretch early potions with blueberry syrup (+90s) and maple syrup (+3 min). Maple mainly useful before offering engine online. Once engine running, potions never a bottleneck.
4
Seeker + Verdant 4pc + Mask → ~26% cedar rare drop rate
Mask+Seeker only = ~14%. Verdant gloves/top/bottom/boots = +80% non-combat rare drops. Exact stacking formula unconfirmed. Verdant from Bloom festival: pieces 1-3 at 20/40/60 currency · gloves = 1% drop · hat skipped (Mask better).
5
Tailoring + fishing → markers → Gilded Satchels → 4 Forgotten gear pieces
250 markers = 1 guaranteed Gilded Satchel (lv5 LK mentee). Only 4 pieces needed — hat slot = Mask always, cape slot = Cryptid Cape for rare mob farming. Save/load: 60-70 satchels sufficient at ~5% rate. Repairs need tailoring + hunting trap materials.
6
Gathering (90%) → tide rush to lv75+
4× XP + 50% speed + 99% high tide = ~4M XP/hr. Use lv82 + gathering potion +8 + mask = effective lv90 for max community mentoring XP locked in. Levels gathering mentor → materials on demand.
7
Gold Rush → gold chain → crafting/smithing/mining XP simultaneously
Gold ore → mining XP · bars → smithing XP · rings → crafting XP · gold armor = day 1 combat set. Jade tools bypass smithing entirely (~42 jade total: 10×4 tools + 2 rings). Seeker + Hydraulic Mining sufficient jade supply.
8
Cooking (60-70%) — can bottleneck at high run speeds
Best speed booster is the lv90 gathering item. Don't neglect cooking allocation at high legacy counts.
